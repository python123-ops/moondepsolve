# Security Policy

## Supported Version

当前维护版本为 `0.2.x`。尚未发布安全维护承诺的旧版本仅接受评估，不保证回移修复。

## Reporting

请优先通过 GitHub Security Advisory 私密报告安全问题：

<https://github.com/python123-ops/moondepsolve/security/advisories/new>

报告请包含受影响 API、最小复现、预期影响和建议修复。若私密通道不可用，请在 GitLink/GitHub 创建不含利用细节的 Issue，请求维护者建立私密沟通。

维护者会确认收到报告、评估影响，并在修复可用后协调披露。不要在修复前公开可直接利用的细节。
